OLIGHTBOX

A super simple lightbox

Learn More Just Download

Usage

Set data-toggle="olightbox" on the trigger element and set data-target="url" or just wrap it with an a-tag. 

<img src="example_small.jpg"
    data-toggle="olightbox" data-target="example_big.jpg">
<!-- or -->
<a href="example_big.jpg" data-toggle="olightbox">
    <img src="example_small.jpg">
</a>
Of course you can always do it the javascript way 
$(".olightbox").olightbox(options);

Customize

Do it like you always do it. 

$(".olightbox").olightbox({
'settings_key' : 'settings_value'
});

Well, this is how the settings looks like. If you change the className-attributes you're gonna break the style though.

Key Value Description Default
transition string transition type:
fadeIn, slide, show etc		 fadeIn
trigger string event type:
hover, click etc		 click
show boolean show box directly on trigger true

Methods

.olightbox(options)

Activate olightbox with optional options object

$(".olightbox").olightbox({show: false});

.olightbox('prepare')

Prepare all the needed elements

$(".olightbox").olightbox("prepare");

.olightbox('show')

Show the lightbox

$(".olightbox").olightbox('show');

.olightbox('remove')

Remove the lightbox

$(".olightbox").olightbox('remove');
